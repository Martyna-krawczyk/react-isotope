# Overview

This webapp combines together React, Isotope, and Flux (Alt) so you can see one way of accomplishing this combination of libraries.



## Local Installation and development

```bash
npm install

# In one console
npm run dev-server

# In another console
npm run start-dev

# Navigate to http://localhost:8080
```
